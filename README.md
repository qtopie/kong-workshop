# kong-workshop

*Workshop for playing or developing kong locally.*

## Setup Locally

```bash
git submodule update --init
cd plugins/go-pluginserver && make && cd -
cd plugins/go && make && cd

docker-compose up
```


## Testing

* Run demo api

```bash
cd examples
go run main.go
```

* Config demo service with go plugins

* Test with `curl`

```bash
curl -v http://127.0.0.1:8000/ping
```
